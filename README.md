# P2P-Chat
Chat based on peer-to-peer approach. The app has chat rooms, register nickname window, create new chat window, chat window itself

## Used technologies:
* UDP broadcasting
* TCP protocol (to communicate between peers)
* Python 2.7
* Tkinter
